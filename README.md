## ⚡ **Cara Menjalankan Program**

Pastikan Anda sudah menginstal *compiler* seperti **GCC**.
Kemudian jalankan perintah berikut di terminal:

```bash
gcc nama_file.c -o hasil
./hasil
```

Contoh:

```bash
gcc hello_world.c -o hello
./hello
```
